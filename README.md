# pissy
A simple python based image slideshow script



## Usage
./pissy.py



## Optional parameter
To set the image source dir
```
--source /path/to/image/folder
```

To set image delay to 5 sec
```
--delay 5
```

 To enable verbose output
```
--verbose
```
